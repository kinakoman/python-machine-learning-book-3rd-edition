# 表記法

訓練データのデータセット(例えば特徴量の種類)は下付き文字

各データセット内のデータ番号は上付き文字

$$
\boldsymbol{x}=
\begin{bmatrix}
x_1^{(1)}&x_2^{(1)}&x_3^{(1)}&\cdots&x_j^{(1)}\\
x_1^{(2)}&x_2^{(2)}&x_3^{(2)}&\cdots&x_j^{(2)}\\
\vdots&\vdots&\vdots&\vdots&\vdots\\
x_1^{(i)}&x_2^{(i)}&x_3^{(i)}&\cdots&x_j^{(i)}\\
\end{bmatrix}
$$

特徴量jのi番目の訓練データは$x_j^i$で示す
